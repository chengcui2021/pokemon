## Build and run the project
Run Backend:
```
cd server
```
Run the database migrations:

```
$ npx prisma migrate dev --name initialize
```
Build the project:

```
$ npm i
```
Run the project:

```
$ npm start
```
Run Frontend:
```
cd client
```
Build the project:

```
$ npm i
```
Run the project:

```
$ npm start
